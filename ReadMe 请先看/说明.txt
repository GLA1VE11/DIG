1、STQ_UI是运行主程序的文件。若想在Pycharm中运行，直接运行本.py文件
即可；
2、STQ数字图像处理是相关的UI界面
     STQ数字图像处理.py是与界面相关的Python程序
3、BMP.py是封装的与位图基本操作、几何变换相关的类；
4、Strength.py是封装的与位图增强操作的类；
5、Sharpen.py是封装的与位图锐化操作的类；
6、Segmentation.py是封装的与位图分割操作的类；

img文件夹下为相关图片以及默认保存路径，其中：
bmp_save.bmp保存位图几何变换的结果；
strength.bmp保存位图增强操作的结果；
sharpen.bmp保存位图锐化操作的结果
segmentation.bmp保存位图分割操作的结果。

所有算法均手写，只使用cv2和plt进行图片的显示。
位图信息的读取是基于python读取位图文件的，没有使用库函数。

由于环境问题，可执行程序没有放在内，故附上操作视频，在视频中
有明确讲解演示。
